# 互動式日文學習系統 - L19至L21教材平台

一個專門為日語學習者打造的、兼具美感與實用性的互動式學習平台。以進學日文讀本 Lesson 19 (假設形/建議)、Lesson 20 (授受表現/原因/逆接)、Lesson 21 (被動形/傳聞推測) 的教材為核心定制。

## 🌟 特色功能
1. **日語聲調優化引擎 (Pitch Accent TTS)**：
   * 整合 Google Wavenet-based TTS 發音，提供標準、正式且具有抑揚頓挫（高低音）的日式標準發音。
   * 具備離線自動降級 (Fallback) 機制，若離線或 API 限制會自動切換為瀏覽器原生語音。
2. **動詞自動變形模擬器 (Verb Conjugator)**：
   * 支援一類、二類、三類動詞，輸入任意原形動詞（如「書く」）即可自動計算並生成 9 種常用活用變形。
   * 每個變形卡片旁皆配有一鍵發音功能，方便學習各變形後的聲調變化。
3. **課文文法探索教室 (Grammar Classroom)**：
   * 整理 L19-L21 所有重點文法，每個例句均提供一鍵發音與繁體中文對照。
4. **台灣人常犯錯誤對決 (Taiwanese Learner Pitfalls)**：
   * 收錄授受表現中容易混淆的助詞（の、を、に）使用限制、受益者視角與禮貌避坑忠告。
5. **互動挑戰與土鍋煮飯閱讀測驗 (Interactive Quiz)**：
   * 包含 15 道精選文法單選與句子重組（拼句卡片遊戲）挑戰。
   * 整合 Lesson 19「土鍋煮飯方法」長文閱讀，點擊單字即可即時浮現中文翻譯，並附帶相關閱讀測驗。

## 🎨 設計美學
* **Glassmorphism 磨砂玻璃風格**：卡片採用高模糊度背景濾鏡與半透明高光邊框，展現現代沉浸式視覺體驗。
* **暗黑科技感配色**：以深邃藍底色搭配極光藍、初櫻粉、螢光綠等重點點綴色，營造高質感色調。
* **極佳的響應式設計 (RWD)**：支援手機、平板、桌面端，排版自動適應。

## 🚀 本地運行方法
本專案為純前端網頁應用（HTML/CSS/JS），無須配置任何本地伺服器或 Node.js 環境：
1. 將專案下載至本地。
2. 直接雙擊 `index.html` 即可在瀏覽器中開啟並開始學習！
