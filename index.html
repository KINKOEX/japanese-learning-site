L19：假設形與勸告

L19 課文

1. 動詞・形容詞「ば形」

一類: う段 → え段 + ば | 二類: る → れば | 三類: すれば / くれば

表示前面的條件成立時，後面就會跟著發生。常用於規律、邏輯成立的假設。

  • 時間があれば、寝ます。
    如果有時間的話，我就會睡覺。
  • 安ければ買います。
    如果便宜的話，我就買。

2. 假設與後續「たら」

動詞: 読んだら | い形: 寒かったら | な形/名: 元氣だったら / 獨身だったら

口語中限制最少、使用範圍最廣的假設。包含「~之後」的確定假設用法。

  • 明日雨が降ったら行きません。
    如果明天下雨的話，我就不去。 (假設)
  • 5時になったら、帰ります。
    到了五點的話（之後），我就回家。 (確定條件)

3. 建議與對策「なら」

常接名詞、形容詞、動詞普通形 + なら

表示針對對方所提的事情，提供自己的意見、建議或判斷。後句多為建議或要求。

  • 日本に行くなら、ドライヤーを買ってきてください。
    如果是要去日本的話，請幫我買吹風機回來。

4. 勸告「~たほうがいい」

肯定: Vた + ほうがいい | 否定: Vない + ほうがいい

給予對方的強烈建議，「最好做某事」或「最好不要做某事」。

  • 早く寝たほうがいいですよ。
    你最好早點睡覺比較好喔。
  • 無理をしないほうがいいと思います。
    我覺得最好不要勉強自己比較好。

L20：授受表現與原因

L20 課文

1. 行為的授受 (動作恩惠)

Vて + あげる (為他人) | てくれる (他人為我) | てもらう (得到幫助)

日語中非常強調「視角」和「恩惠轉移」。他人為自己做動作，一定要表達感謝。

  • 田中さんは私に本を貸してくれました。
    田中先生借書給我。 (我心存感激)
  • 私は田中さんに作文を直してもらいました。
    我請田中先生幫忙修改了作文。 (受益者視角)

2. 表示原因的「て形」

動詞/形容詞/名詞 + て形 (後句必須是非意志表現)

表示因為前面的情況而引起後面的狀態，後句必須是生理現象、無法控制的事情或遺憾心情。

  • ゆうべは暑くてよく眠れませんでした。
    昨晚因為太熱了，所以沒能睡得好。

3. 逆接與驚訝「のに」

普通體 (な形/名詞 + なのに) + のに

表示轉折，翻譯為「雖然...卻...」。多帶有說話者主觀的「意外、驚訝、不滿或生氣」等情緒。

  • 一生懸命勉強したのに、N1に合格できなかった。
    雖然拼命學習了，卻沒能通過 N1。 (不甘心/失望)

4. 發現的「たら」

Vたら + 過去式句尾 (た)

表示「做了 S1 動作後，發現了 S2 意外的現象或結果」。

  • かばんを開けたら、財布がなくなっていました。
    一打開包包，才發現皮夾不見了。

L21：被動語態與推測

L21 課文

1. 被動語態 (受身形)

一類: a段 + れる | 二類: られる | 三類: される / こられる

表示主體受到他人的影響或動作。分為直接被動、間接被害被動（感到困擾）和無生物主體被動。

  • 私は先生にほめられました。
    我被老師稱讚了。 (直接被動)
  • 雨に降られて風邪を引きました。
    被雨淋了（淋雨受害）而感冒了。 (被害被動)
  • 日本の車はいろいろな国へ輸出されています。
    日本車正向許多國家出口。 (無生物主語客觀描述)

2. 間接轉述「ようにと言う」

Vる / Vない + ように + 言う/頼む/注意する

用來間接轉述第三者指示、要求或警告的事情。類似英語的 indirect command。

  • 医者は主人にお酒を飲まないようにと注意しました。
    醫生叮囑丈夫不要喝酒。

3. 傳聞與推測「らしい」

普通形 (名詞/な形不加だ) + らしいです

表示基於客觀客觀的事實或傳言所做出的合乎情理的推測。「似乎」、「聽說」。

  • 消防車が走っています。どこかで火事があったらしいです。
    消防車在跑，好像什麼地方失火了。

L22：使役形與使役被動

L22 課文

1. 使役形變形與意涵

一類: a段 + せる | 二類: させる | 三類: させる / こさせる

表示「讓、叫、逼」他人做某個動作。主要有四個光譜：強制（逼迫）、允許（成全）、放任（不管）與情感引起。

  • 先生は学生に宿題をさせました。
    老師讓（逼）學生做作業。 (強制)
  • 娘にピアノを習わせます。
    讓女兒學鋼琴。 (成全/允許)

2. 使役句的助詞規則

自動詞：使役主 は 被使役者 【を】 V使役 | 他動詞：使役主 は 被使役者 【に】 賓語 【を】 V使役

自動詞使役中，被使役者用「を」；他動詞使役中，為了避免句子裡出現兩個「を」（雙を禁忌），被使役者必須改用「に」。

  • 母は子どもを買い物に行かせました。
    媽媽讓小孩去買東西。 (行く是自動詞，用 を)
  • 先生は学生に作文を書かせました。
    老師讓學生寫作文。 (書く是他動詞，用 に...を)

3. 請求允許「～させていただけませんか」

V使役て形 + いただけませんか (極具禮貌的職場請求許可句)

表示「能不能請您讓我做某事？」。比一般的請求更能展現謙遜和禮貌，常用於請假或提出提議。

  • 熱があるので、今日休ませていただけませんか。
    因為發燒，今天可以讓我請假嗎？

4. 使役被動形（被迫/被害感）

一類: a段 + される (す結尾為 させられる) | 二類: させられる | 三類: させられる / こさせられる

表示「被迫、不得不做某事」，通常帶有很不情願、覺得麻煩的負面心情。

  • 私は母に野菜を食べさせられました。
    我被媽媽逼著吃蔬菜。 (很不甘願)
  • 昨日、私は先輩にお酒を飲まされました。
    昨天我被前輩逼著喝酒了。 (一類動詞飲まされる)

痛點 1：授受表現中「助詞」大不同！

L20 授受補充

中文我們習慣都用「幫給某人做...」，導致學生會一律套用助詞 **に**。但在日語中，助詞的選用完全取決於**動詞**的性質：

助詞 適用情境 常見動詞 經典例句
の 動作的對象是接受者的**身體部位**或**所有物** 手伝う、直す、洗う、運ぶ、持つ 弟の宿題を手伝ってあげる。 (手提/輔助所有物)
を 動作的直接對象**是人**（及物動詞） 送る、招待する、呼ぶ、連れて行く、しかる 留學生をパーティーに呼んであげる。 (人是受詞)
常犯錯誤
弟に宿題を手伝ってあげました。

直接用「に」表示幫給弟弟做事，是生硬的中式日文。

正確表達
弟の宿題を手伝ってあげました。

因為手傳的是弟弟的「宿題」（所有物），所以使用「の」。

痛點 2：忘記轉換為「受益者視角」(てくれる / てもらう)

學習心態轉折

中文是「客觀事实描述」的語言，而日文則是「情感連動與感恩」的語言。台灣同學在描述別人幫自己做事時，常常只說客觀動詞（作りました），這在日語中聽起來是冷淡甚至失禮的。

受益視角核心觀念
  • ～てくれた：主體是他人，強調「他人體貼我，為我做了事」。
  • ～てもらった：主體是我，強調「我請求並得到了他人的幫助」。
常犯錯誤 (語氣冷淡)
山田さんは料理を作りました。

聽起來像是在匯報「山田煮了飯」的客觀事實，沒有半點感激之意。

正確表達 (富有人情味)
山田さんは料理を作ってくれました。

帶有濃濃的感謝心意：山田先生（特地）為我煮了料理！

痛點 3：對上司、長輩用「てあげる」(極其失禮！)

台生高頻雷區

「あげる / てあげる」暗示著一種 **上對下**、施捨恩惠的語氣。如果您對老師或老闆說這個詞，會讓人有一種「我在恩賜你、為你施恩並要求你感謝」的傲慢感。

禮貌替代公式：
  • 改用提議語氣：～ましょうか (我來...好嗎？)
  • 改用謙讓語結構：お + 動詞ます形去ます + します
長輩面前禁用 (失禮/高傲)
先生、荷物を持ってあげます。

潛台詞：「老師，我施恩幫你提行李囉，感恩我吧！」

正確表達 (謙遜優美)
先生、荷物をお持ちします。

使用「お持ちします」(謙讓表現)，謙虛地表達自己願意為老師分擔重物。

痛點 4：使役句中絕對不能出現兩個 を！(雙を禁忌)

L22 助詞陷阱

在日語中，一個單句（沒有從屬子句）中一般不允許出現兩個由助詞 **を** 引導的賓語。因此在使役句中：

動詞類型 被使役者助詞 理由 經典例句
自動詞 (如 行く) を 自動詞本身沒有賓語，不會衝突。 学生を行かせます。
他動詞 (如 書く) に 他動詞已用「を」接賓語，為避開「雙を」，人必須改用「に」。 学生に作文を書かせます。
常犯錯誤 (雙を衝突)
先生は学生を作文を書かせました。

一個句子出現了兩個「を」，違反日語語感，非常彆扭。

正確表達 (避開雙を)
先生は学生に作文を書かせました。

將人（學生）的助詞改為「に」，完美避開「雙收」禁忌！

痛點 5：中文「讓」的無禮使役陷阱

職場與日常禁忌

中文我們經常說「讓我來做...」或「讓老師教我...」。但如果直接把中文的「讓」翻譯成日文的使役形（如：先生に教えさせます），聽在日本人耳裡，會變成**「強迫、逼迫老師教我」**，非常無禮與傲慢！

正確的表達思維：
  • 請求別人為自己做某事：改用恩惠授受 **～てもらう** 或 **～ていただく**。
  • 請求別人「允許」自己做某事：改用使役加恩惠 **～させてくれる/～させてもらう**。
中文直譯 (極度失禮！)
私は先生に日本語を教えさせました。

聽起來像：「我命令、逼迫老師教我日語」（上對下的強制）。

正確表達 (感恩受恩)
私は先生に日本語を教えていただきました。

使用「Vていただく」(請人做某事)，表示感謝老師屈尊教導我。

土なべでご飯を炊く方法

土で作ったなべ「土なべ」でご飯を炊く方法を紹介しましょう。 土なべで炊くと、電気釜よりずっとおいしいごはんができます。

まず、米２合(※１)を洗い、ざるに入れて、水が切れる(※３)まで 30 分待ちます。 それから、土なべに米を入れて、水を 400 ㏄加えます。平らなふたを先に置き、それから大きいふたをして、そのまま置いておきます。

20 分たったらガスに火をつけます。火は普通の強さにして、15 分たったら、白い湯気が出ているのを確認して、火を止めます。 そして、そのまま置いておきましょう。20 分後にできあがりです。途中でふたを開けて中を見てはいけません。できあがったら、ほかの入れ物に移しましょう。

柔らかいご飯がいい場合は、少しだけ水の量を増やしてください。ただし、時間は決められたとおりにしてください。

補充註釋：
(※1) 合(ごう)：米或酒的度量單位，1合＝180 cc
(※2) ざる：有網眼的竹篩、瀝水籃架
(※3) 水が切れる：水分完全濾乾、不殘留水

時間線對照梳理：

  • 洗米並濾乾：30分鐘
  • 加水靜置：20分鐘 (蓋上兩層蓋子)
  • 開中火煮飯：15分鐘 (直至冒白氣)
  • 關火悶飯：20分鐘 (中途切忌開蓋)
  • 總耗時：約1小時25分

閱讀考點提醒：

此文章考察對「假設與步驟」的關聯。例如問句「飯煮好後，以下做法何者不符？」，正確答案為「不能一直把飯留在土鍋裡 (移しましょう)」，悶飯時「不能中途開蓋子」等。
